# Travel Explorer 🌍

Welcome to **Travel Explorer**, a premium, modern, and highly responsive single-page web application designed for travel enthusiasts looking to explore the world and book unforgettable journeys.

🔗 **Live Website URL:** [https://btit25asgt-hub.github.io/Travel-Explorer/](https://btit25asgt-hub.github.io/Travel-Explorer/)

---

## 📖 What This Website Is About
**Travel Explorer** is a travel agency front-page designed to provide users with a clean, engaging, and elegant platform to discover travel destinations, browse vacation packages, book trips, and contact support. Built with a responsive layout, modern typography, glassmorphism elements, and smooth interactions, it provides a seamless user experience.

---

## ⚡ Features & What It Can Do
* **Interactive Destination Search:** Users can filter through popular destinations (Paris, Tokyo, Nepal, Dubai, Bali, Switzerland) dynamically in real-time as they type in the search bar.
* **Curated Travel Packages:** Clear pricing structure, details, and features for various budgets (Basic, Standard, and Premium packages).
* **Smooth Auto-Scrolling:** Clicking any "Book Now" button automatically scrolls the page smoothly to the booking form.
* **Dynamic Booking Form:** A fully validated form that lets users select destinations, dates, packages, and submit their requests, displaying a timed success notification upon submission.
* **Contact Support Form:** An integrated customer service form with success notification popups on submission.
* **Fully Responsive Design:** Optimized for all screen sizes, including mobile phones, tablets, and desktops using Bootstrap 5.3 and custom CSS media queries.
